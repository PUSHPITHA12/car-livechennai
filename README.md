# 🚗 Live Chennai Cars

A modern used car marketplace website focused on Chennai. Users can browse cars, compare models, view detailed specifications, and explore popular car brands through a clean and responsive interface.

## 🌐 Live Demo

https://pushpitha12.github.io/car-livechennai/

## ✨ Features

* Browse used cars
* Detailed car information
* Car comparison page
* Brand-wise car listings
* Responsive design
* Chennai-focused marketplace
* Interactive image galleries
* Modern red-and-white UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## 📂 Project Structure

* index.html – Home Page
* cars.html – Car Listings
* details.html – Car Details
* compare.html – Compare Cars
* brands.html – Brand Listings
* models.html – Car Models
* about.html – About Us
* contact.html – Contact Page

## 🎯 Project Objective

To provide a user-friendly platform for exploring and comparing used cars available in Chennai while delivering a professional marketplace experience.
